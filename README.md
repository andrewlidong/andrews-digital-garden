# Andrew's Digital Garden

My personal website featuring a terminal-inspired interface. Visit at [andrewlidong.xyz](https://andrewlidong.xyz).

## Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build and deploy
npm run deploy
```
