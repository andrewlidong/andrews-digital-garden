# Trace and Pace

## A Strava art run club

Trace and Pace is a Strava art run club — runners trace shapes and drawings onto the city with their GPS routes, then meet up to share them.

### Details
- **Where:** [Georgie's Cafe](https://www.georgies.cafe/), Lower East Side
- **When:** Sundays at 11am
- **Strava:** [strava.com/clubs/traceandpace](https://www.strava.com/clubs/traceandpace)

### Technologies Used
- HTML
- CSS
- Cloudflare Pages

### Development Process
The site is a static HTML landing page at [traceandpace.com](https://traceandpace.com), deployed on Cloudflare Pages — every push to `main` auto-deploys. It keeps things deliberately simple: a clear hero, the meeting details, and a link out to the Strava club.
